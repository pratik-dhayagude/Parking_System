🚗 Smart Parking System

📌 Project Overview

Smart Parking System is a Java-based Parking Management Application designed using Object-Oriented Programming (OOP) principles. The system automates vehicle entry, parking allocation, ticket generation, billing calculation, and vehicle exit management, making parking operations efficient and organized.

⸻

🌟 Key Features

🚘 Vehicle Management

	•	Supports multiple vehicle types:
	•	Car
	•	Bike
	•	Truck
	•	Stores vehicle registration numbers.


🎫 Automatic Ticket Generation

	•	Generates a unique parking ticket when a vehicle enters.
	•	Records entry time automatically.

🅿️ Smart Parking Allocation

	•	Finds and assigns available parking spots.
	•	Prevents duplicate parking.
	•	Tracks occupied and vacant spots.

⏱️ Parking Duration Tracking

	•	Calculates parking time using Java Date & Time API.
	•	Tracks exact entry and exit timestamps.

💰 Billing System

	•	Calculates parking charges based on parking duration.
	•	Supports hourly billing logic.
	•	Minimum billing for first hour.
